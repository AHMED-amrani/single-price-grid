# Single Price Grid Component

Frontend Mentor Project: Single Price Grid Component

## Technologies Used
- HTML
- CSS

## Project Description
This project is a solution for the Frontend Mentor challenge: Single Price Grid Component.  
The project includes the following sections:
- **Join Our Community**: Main heading with a brief introductory text.  
- **Monthly Subscription**: Subscription plan with price and a sign-up button.  
- **Why Us**: Section highlighting the benefits and features of the subscription.
